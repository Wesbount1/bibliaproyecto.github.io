# 📖 Biblia (Reina Valera 1960) – Proyecto Estático

Una interfaz web minimalista y rápida para consultar la **Santa Biblia (RVR1960)**. Este proyecto nace con el fin de ofrecer una navegación fluida entre libros y capítulos, optimizada para la lectura y el estudio personal.

## ✨ Características principaless
- **Navegación Intuitiva:** Acceso rápido a los libros y capítulos disponibles.
- **Arquitectura Estática:** Carga ultra rápida sin necesidad de bases de datos complejas.

## 🚀 Tecnologías utilizadas
- **HTML5 / CSS3** (Interfaz limpia y responsiva)

## ⚠️ Nota del desarrollador (Estado del Proyecto)
Este proyecto se encuentra actualmente en fase **Beta**. 
> **Importante:** No todos los libros han sido agregados aún. Se están integrando progresivamente para asegurar la integridad del texto.
